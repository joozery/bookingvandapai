# 🚐 BookingVan Real-Time: ระบบจองที่นั่งรถตู้ท่องเที่ยวครบวงจร

ระบบจองที่นั่งรถตู้ท่องเที่ยวแบบเรียลไทม์ (Real-time Booking) พัฒนาด้วย Next.js, Tailwind CSS v4, และ TypeScript ครบเครื่องทั้งระบบจองตั๋วของลูกค้าและแผงควบคุมระบบของแอดมิน ปรับแต่งมาเพื่อการใช้งานบนมือถืออย่างยอดเยี่ยม!

## 🚀 วิธีการรันและทดสอบระบบงาน

1. **ติดตั้ง dependencies (หากต้องการติดตั้งใหม่):**
   ```bash
   npm install
   ```
2. **รัน Development Server:**
   ```bash
   npm run dev
   ```
   จากนั้นเปิดเว็บบราวเซอร์ไปที่ [http://localhost:3000](http://localhost:3000)

## 🏗️ โครงสร้างสถาปัตยกรรมของระบบ (System Architecture)

ระบบประกอบไปด้วย 4 ระบบหลักที่เชื่อมโยงถึงกันอย่างสมบูรณ์แบบ:
1. **ระบบทริป (Trip System):** เพิ่ม แก้ไข ลบทริป กำหนดราคา วันเดินทาง เวลาขึ้นรถ และจุดขึ้นรถ
2. **ระบบรถตู้ (Van System):** บริหารจัดการจำนวนตู้รถตู้ของแต่ละทริป กำหนดป้ายทะเบียน ข้อมูลคนขับ และบริหารจัดการชื่อสตาฟประจำรถ
3. **ระบบจองที่นั่ง (Seat Booking System):** จองที่นั่งแบบเรียลไทม์ (ซิงค์ข้อมูลทุกๆ 2 วินาที) ล็อกสถานะเบาะเพื่อรอแอดมินอนุมัติ และรองรับการขอย้ายที่นั่งโดยที่เบาะเก่ายังไม่หลุดจนกว่าจะได้การอนุมัติที่นั่งใหม่
4. **ระบบเช็คอิน (Check-in System):** ตั๋วเดินทางดิจิทัลพร้อมรหัสและคิวอาร์โค้ดแสดงรายละเอียดการเดินทางครบครัน พร้อมระบบกล้องสแกนคิวอาร์จำลองสำหรับสตาฟ/คนขับเพื่อเช็คชื่อขึ้นรถ

## 🎨 ระบบสีของเบาะนั่ง (Seat Color Legend)

* 🟢 **สีเขียว = ว่าง (Available)** - ลูกค้าสามารถกดเลือกเพื่อส่งคำขอจองได้
* 🟡 **สีเหลือง = รออนุมัติ (Pending)** - ที่นั่งถูกล็อกไว้เพื่อรอแอดมินกดยืนยันการจอง ป้องกันการกดซ้ำซ้อน
* 🔴 **สีแดง = จองแล้ว (Booked)** - ที่นั่งได้รับการอนุมัติการจองแล้ว
* ⚫ **สีดำ = คนขับรถ (Driver)** - ที่นั่งคนขับประจำรถตู้
* 🟣 **สีม่วง = สตาฟ (Staff)** - ที่นั่งสำหรับสตาฟนำทริป สามารถคลิกแก้ไขชื่อสตาฟในระบบจัดการได้

## 📂 โครงสร้างโฟลเดอร์ของโครงการ

```bash
bookingvan/
├── src/
│   ├── data/
│   │   └── db.json          # ฐานข้อมูลตัวอย่าง (Trips, Vans, Bookings)
│   ├── lib/
│   │   └── db.ts            # ตัวช่วยความเสถียรข้อมูลและการจัดทำ Mock Data
│   ├── app/
│   │   ├── api/
│   │   │   ├── trips/       # API จัดการข้อมูลทริปเดินทาง
│   │   │   ├── vans/        # API จัดการข้อมูลรถตู้และคนขับ
│   │   │   ├── bookings/    # API ควบคุมการจองและอนุมัติที่นั่ง
│   │   │   └── db-reset/    # API รีเซ็ตข้อมูลจำลองสำหรับแอดมิน
│   │   ├── admin/
│   │   │   └── page.tsx     # แผงแดชบอร์ดควบคุมของแอดมิน (หลังบ้าน)
│   │   ├── page.tsx         # หน้าจอจองและตั๋วของลูกค้า (หน้าบ้าน)
│   │   ├── layout.tsx       # การตั้งค่า SEO เมทาดาต้าและแบบอักษร Kanit
│   │   └── globals.css      # สไตล์หลักและแอนิเมชัน
│   └── package.json
└── README.md                # คู่มือการใช้งานระบบ
```
